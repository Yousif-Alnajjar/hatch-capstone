# Photo drop-in

Place your Miata hero photograph here as **`miata-hero.jpg`**.

Recommended:
- At least 2400 × 1600 px (horizontal orientation).
- Roof-down, three-quarter front, natural light.
- JPEG, 80–90% quality, keep under ~600 KB.

The article's hero layout automatically picks up `/assets/photos/miata-hero.jpg`.
If the file is missing, the hero area falls back to a typographic title card.
